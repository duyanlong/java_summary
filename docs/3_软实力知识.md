### 情感账户
如果我们给两个人的关系开通一个银行账户，那么当我们经营这段关系时，就是在往账户里面存钱；而当
我们犯了错，或者做了伤害对方的事情，就相当于从账户中取钱。
所谓情感账户，储存的是增进人际关系不可或缺的“信赖”，也就是他人与你相处时的一份“安全感”。
能够增加情感账户存款的，是礼貌、诚实、仁慈与信用。这就是情感账户，如果你们能定期给彼此的情感
账户充值，那么你们之间的银行账户余额就会越来越多，彼此的信任度也会越来越高。
充值方式：
1. 理解他人，理解他人是一切感情的基础。只有了解并真心接纳对方，才可以增进彼此的关系。
2. 注意小节，一
   些看似无关紧要的小节，如疏忽礼貌，不经意的失言，最能消耗情感账户的存款。在人际关系中，最
   重要的正是这些小事。
3. 信守承诺， 个人的信用是一笔很大的资产，如果失信的话，所付出的代价可能远超你的想像。一次严重的失信使
   人信誉扫地，再难建立起良好的互赖关系。
4. 明确期望， 在人与人相处的过程中，我们要明确提出自己的期望，让相关的人都了解到。如果没有明确的期望，人
   们就会变得感情用事，原本简单的小误会也会变得很复杂，原本很小的事情也会导致严重的冲突和人身
   攻击，最终不欢而散。
5. 正直诚信， 正直诚信能够产生信任，也是其他感情投资的基础。诚信，即诚实守信，既要有一说一，又要信守承诺、
   履行约定，而表现这种品格的最好方法就是避免背后攻击他人。
